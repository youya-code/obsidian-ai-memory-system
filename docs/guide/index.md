# 搭建指南

本指南带你逐步完成 Claude Code + Obsidian 记忆系统的搭建。

## 你将获得

搭建完成后，你的 AI 编程助手将拥有：

- 🧠 **持久化记忆** — 约定、踩坑、偏好永不丢失
- 🔍 **按需检索** — 记忆按标签加载，不浪费 token
- ☁️ **跨机器同步** — 通过 git 或 Obsidian Sync 迁移
- 👁️ **完全可视** — 所有记忆在 Obsidian 中随时查看编辑

## 四步搭建

1. [安装 Obsidian](/guide/install-obsidian) — 安装笔记软件 + 必需插件
2. [配置 MCP 连接](/guide/config-mcp) — 让 Claude Code 能读写 Obsidian
3. [创建记忆体系](/guide/create-memory-system) — 建立文件夹 + 标签 + 指针
4. [使用模板](/guide/templates) — 下载现成模板快速上手
